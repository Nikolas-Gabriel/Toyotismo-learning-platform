# Quiz Toyotismo 🚀

Projeto Web interativo desenvolvido para a disciplina de Administração Básica no IFMT.

## 🛠 Tecnologias Utilizadas
- **HTML5 / CSS3** (Interface Responsiva)
- **JavaScript (ES6+)** (Lógica do Quiz e Cronômetro)
- **Firebase Firestore** (Banco de dados em tempo real para Ranking)
- **Chart.js** (Gráficos de desempenho)

## 📌 Funcionalidades
- Escolha de turma para filtragem de resultados.
- Cronômetro em tempo real.
- Ranking Global e por Turma.
- Gráfico comparativo de acertos.
